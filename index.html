Google Cloud の設定

Google カレンダーと連携するために
OAuth クライアントIDが必要です。

取得方法を見る →
  1. Google Cloud Console を開く
  2. 新しいプロジェクトを作成
  3. 「APIとサービス」→「認証情報」→「認証情報を作成」→「OAuthクライアントID」
  4. アプリの種類：「ウェブアプリケーション」
  5. 承認済みのJavaScriptオリジンに http://localhost:8080 と後でGitHub PagesのURLを追加
  6. Google Sheets API と Google Calendar API を有効化
